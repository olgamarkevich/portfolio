# Portfolio Маркевич Ольги

## Скрипты package.json:

| Скрипт | Назначение |
| ------ | ------ |
| dev | Запустит webpack-dev-server с _горячей_ заменой модулей |
| build | Соберет проект для **production** (проект готов к загрузке на сервер) |

#### Чтобы запустить скрипт:
```sh
$ npm run имя_скрипта
```
